# 🎓 StudyTrail

StudyTrail is a full-stack social networking platform for students where users can connect, share posts, like, and comment—similar to LinkedIn but designed for students.

## 🚀 Features

* User Authentication (Register/Login)
* User Profile
* Create Posts
* Like Posts
* Comment on Posts
* Infinite Scrolling Feed
* View Other User Profiles
* Connection System
* Responsive Design

## 🛠️ Tech Stack

### Frontend

* React
* Vite
* Tailwind CSS
* Axios
* React Router

### Backend

* Node.js
* Express.js
* MongoDB
* Mongoose
* JWT Authentication
